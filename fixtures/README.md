# Fixture Notes

Most files under `fixtures/minimal/` are generated by `tools/generate_fixtures.py`.

Large real-world example files are not part of source control. Keep those in
`fixtures/local/` for optional local regression and differential testing. The
CLI contract suite will skip the corresponding real-camera tests when those
files are absent.

One fixture is intentionally vendored from a real-world sample corpus:

- `fixtures/minimal/real_exif.heic`
  - Source: `image4.heic` from [`dsoprea/heic-exif-samples`](https://github.com/dsoprea/heic-exif-samples)
  - Purpose: ExifTool-backed HEIF differential coverage against a spec-faithful sample
  - Upstream license: MIT
